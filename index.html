HTML

HTML을 통해 웹페이지의 기본 구조를 만들 수 있습니다. 콘텐츠를 논리적으로 구분하고 의미를 부여해 접근성 높은 페이지를 설계할 수 있습니다.

CSS

06

JavaScript

07

jQuery

jQuery를 통해 복잡한 JavaScript 기능을 간단하게 구현할 수 있습니다. DOM 조작, 애니메이션, 이벤트 제어를 손쉽게 다룰 수 있습니다.

08

GitHub

Git과 GitHub를 통해 디자인 및 코드의 버전을 체계적으로 관리할 수 있습니다. 팀 프로젝트에서 함께 작업하며 유지보수를 원활히 진행합니다.

09

React

React를 통해 컴포넌트 단위로 디자인을 구조화할 수 있습니다. 디자인 변경 시에도 유지보수가 쉬운 구조적 스타일 시스템을 만들 수 있습니다.
